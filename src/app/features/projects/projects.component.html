Projects

None of these projects are fully mature or have more than a few hours of work put into them, not much more than wireframes. They have been assembled for this showcase recently.
  • PDFutils - Java + Maven command line utility to extract x509 certs from PDFs.
  • Menu Maker - Angular + Spring Boot application to build menus and requisitions for an institutional setting.
  • MsgToEmlConverter - Java + Maven command line utility to convert MSG to EML format.
  • This Showcase - Angular + Node/Express
  • More on the way.
